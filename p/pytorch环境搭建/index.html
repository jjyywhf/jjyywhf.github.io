Pytorch环境搭建

2026-05-26
阅读时长: 2 分钟

本文使用conda+VScode，搭建pytorch开发环境

总述：核心三件套&自选两件套

整个环境当中，最关键的是三个部分：

  • python：语言，因为pytorch是用python写的
  • pytorch：机器学习库，由Facebook开发，全球都用
  • cuda：nVidia专有的工具，用GPU加速简单并行计算，比CPU快上百倍（这就是N卡nb的原因） 注意：以上三个部分的版本必须搭配！！！

还有两个部分，取决于个人喜欢：

  • anaconda：环境管理器+包管理器，用起来很方便就是有点大，推荐使用
  • VScode：写代码的地方，这个完全随意

STEP1：安装python

这步直接跳过: )

STEP2：用conda创建虚拟环境

（1）下载anaconda

下载地址：conda官网 不想注册的话，下面有一个小小的skip~

进入第二个页面，windows系统直接点第一个

（2）安装anaconda

这步选这三个，环境变量我们自己配。其他一路跟着引导走就行，安装的比较慢，需要多等一会。

安装完成后把这5个东西添加到环境变量里，这里提供相对路径作为参考，具体取决于安装位置

打开终端，输入conda -V，如果显示出版本说明安装成功！

（3）使用conda创建并激活虚拟环境

在终端中，输入conda create -n test python=3.11 -y 这个命令就是在创建一个3.11版本的名叫test的环境 其中-n后面是名称，python=后面是版本，目前建议3.11

然后输入conda activate test 这个命令就是激活刚刚创建的环境，完成后命令行前面会有一个括号test

（4）在虚拟环境中安装pytorch

首先我们要确认一下cuda的版本，在终端中输入nvidia-smi 可以在右上角看到我这里是13.2

然后来到pytorch官网，按照实际情况选择，并把它给出的命令输入到终端

等一会就安装成功了。

（5）在VScode中打开虚拟环境

在右下角有个显示版本的位置，就是后面带（py311——我的虚拟环境名称）

然后选择刚才创建的就可以了

import torch

print(torch.__version__)
